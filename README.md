## How It Works

1. The user selects an image file from the system  
2. The application reads and processes the selected image  
3. The image is uploaded and stored in the assets directory  
4. The uploaded image can be previewed or used further in the application  

This project keeps the logic simple and avoids unnecessary complexity.
